# 🎮 Text-Based RPG Game

## Overview

This is a turn-based RPG game developed in Java for a programming hackathon. Players battle increasingly difficult enemies, earn points, purchase upgrades, and attempt to survive all 10 rounds.

## Features

- ⚔️ Turn-based combat
- ❤️ Healing system with limited uses
- 📈 Enemy difficulty scaling
- 🛒 Upgrade shop between rounds
- 🎯 Critical hit and dodge mechanics
- 🏆 Score tracking

## Technologies

- Java
- Object-Oriented Programming
- Random number generation
- Console interface

## How to Run
Open `RPG.java` in your preferred Java IDE. Then, compile and run the program.
